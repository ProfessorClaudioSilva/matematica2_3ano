O que é HTML?

HTML significa HyperText Markup Language ou Linguagem de Marcação de Hipertexto. É a linguagem padrão usada para criar páginas da web.

Estrutura Básica de um Documento HTML

Todo documento HTML possui uma estrutura básica. Veja o exemplo abaixo: 


<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Título da Página</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
    <p>Este é um parágrafo.</p>
</body>
</html>

Principais Tags HTML

  • <h1> até <h6> - Títulos e subtítulos
  • <p> - Parágrafo
  • <a> - Link
  • <img> - Imagem
  • <ul>, <ol>, <li> - Listas

<h1>Título Principal</h1>
<p>Este é um texto de exemplo.</p>
<a href="https://www.exemplo.com">Clique aqui</a>
<img src="imagem.jpg" alt="Descrição da imagem">
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

Exemplo Prático

Veja como criar uma página simples:


<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Minha Primeira Página</title>
</head>
<body>
    <h1>Bem-vindo ao HTML!</h1>
    <p>Essa é minha primeira página da web.</p>
    <a href="https://www.w3schools.com">Aprenda mais</a>
</body>
</html>
